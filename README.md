# Trapping Rain Water Problem

## Overview

This project solves the classic Trapping Rain Water problem using Python.

Given an array representing the height of bars, the program calculates how much rainwater can be trapped between the bars after rainfall.

---

## Problem Statement

Given an array of non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.

### Example

Input:

```python
height = [3, 0, 2, 0, 4]
output:7
